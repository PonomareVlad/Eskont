Noted / Important

  1. Loop & Counter

    Wont work well with swiper: loop = true; & photoswipe: counterEl: true,(What is counter? example: 1/5...2/5) - "loop" duplicate images - the photoswipe counter will be wrong. *** If you dont want a loop - you can set photoswipe counter counterEl: true,

  2. Markup

    Schema.org markup + semantic HTML: use unordered (bulleted) list (If you want a div change JS - "(find) control+f-->" tagname value) . Copy-paste - this code to check: Structured Data Testing Tool - Google

  3. Match index

    Extra CODE "match index" - EXAMPLE: When you click(zoom) image1 -- goes to image 2 - close image2 (X) - also the swiper update is position (BETTER User Experience) (find(ctr +f)--> mySwiper.slideTo(getCurrentIndex, false);) - This idea miss in most slider & lightbox examples/plugins mixed.

  4. Photoswipe options

    JS - line (find) -ctr +f --> the term:// define options (if needed). You find endless options for photoswipe - This is the place to add/modify options. Full Options list her PhotoSwipe Options

  5. SWIPER options

    slideperview

    slideperview - option1: Set number (1,2,3 and so on) - example ||||| option2("Carousel Mode" this example): Set to "auto" than add CSS width Property .swiper-slide (in thie case eash slide is 88% width) - example.
